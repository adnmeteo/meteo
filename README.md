## Station météo amateur à Thonon-les-Bains

Site de données météo, station météo.

Les observations sont réalisées avec une station Bresser 5-en-1 installée sur un balcon au 4ᵉ étage, au centre-ville de Thonon-les-Bains. Plusieurs améliorations ont été apportées afin de limiter autant que possible l'influence du rayonnement solaire et de l'îlot de chaleur urbain sur les mesures de température.
